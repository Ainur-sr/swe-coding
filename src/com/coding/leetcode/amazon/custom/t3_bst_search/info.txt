Обход дерева
Задача: Дано бинарное дерево. Реализуйте четыре типа обхода:

In-order (левый → корень → правый)
Pre-order (корень → левый → правый)
Post-order (левый → правый → корень)
Level-order (по уровням, слева направо)
Пример:

Дерево:     1
           / \
          2   3
         / \
        4   5

In-order: [4, 2, 5, 1, 3]
Pre-order: [1, 2, 4, 5, 3]
Post-order: [4, 5, 2, 3, 1]
Level-order: [1, 2, 3, 4, 5]
