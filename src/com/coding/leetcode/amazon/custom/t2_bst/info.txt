Базовое BST - Вставка, поиск, удаление
Задача: Реализуйте класс Binary Search Tree со следующими методами:

insert(int value) - вставить значение в дерево
search(int value) - найти значение в дереве, вернуть true/false
delete(int value) - удалить узел с заданным значением
Пример:

Input: insert(5), insert(3), insert(7), search(3), delete(3), search(3)
Output: true, false
